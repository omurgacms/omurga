# Code of Conduct

Omurga CMS aims to be a respectful and useful open-source project.

## Expected behavior

- Be respectful.
- Give constructive feedback.
- Keep discussions focused on the project.
- Help new contributors when possible.

## Unacceptable behavior

- Harassment or personal attacks
- Insults, threats or discriminatory language
- Spam or unrelated promotion
- Sharing private information without permission

## Reporting

If you see behavior that harms the project community, please report it through the repository maintainers.

# Davranış Kuralları

Omurga CMS saygılı ve faydalı bir açık kaynak projesi olmayı hedefler.

## Beklenen davranışlar

- Saygılı olun.
- Yapıcı geri bildirim verin.
- Tartışmaları proje odağında tutun.
- Yeni katkı verenlere yardımcı olun.

## Kabul edilmeyen davranışlar

- Taciz veya kişisel saldırı
- Hakaret, tehdit veya ayrımcı dil
- Spam veya alakasız tanıtım
- İzin olmadan özel bilgi paylaşımı

## Bildirim

Projeye zarar veren bir davranış görürseniz repo yöneticilerine bildirin.
