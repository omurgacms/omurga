# Topluluk V1

Omurga CMS için dernek, vakıf, platform ve topluluk sitelerine yönelik tema.

## Panel

`/themes/topluluk-v1/admin/panel.php`

Tema aktifken admin menüde **Topluluk V1 Paneli** olarak görünür.

## Demo mantığı

Demo yükleme sayfalar, kategoriler, haber/duyuru içerikleri, menüler ve tema görünüm ayarlarını hazırlar.
Tema silinirse yazılar, sayfalar, kategoriler, etiketler, görseller ve menüler silinmez.
