# Security Policy / Güvenlik Politikası

## English

Omurga CMS is currently in beta. Please do not publicly disclose security vulnerabilities before maintainers have time to investigate.

### Reporting a vulnerability

Send a private report to the project maintainers with:

- Affected version
- Steps to reproduce
- Expected and actual behavior
- Screenshots or logs if available
- Suggested fix, if known

### Supported versions

| Version | Status |
|---|---|
| 1.1.0 Beta | Supported beta |
| Older beta versions | Upgrade recommended |

---

## Türkçe

Omurga CMS şu anda beta aşamasındadır. Güvenlik açıklarını, geliştiriciler inceleme fırsatı bulmadan herkese açık şekilde paylaşmayın.

### Güvenlik açığı bildirimi

Proje geliştiricilerine özel olarak şu bilgilerle bildirim yapın:

- Etkilenen sürüm
- Tekrarlama adımları
- Beklenen ve gerçekleşen davranış
- Varsa ekran görüntüsü veya log
- Biliniyorsa çözüm önerisi

### Desteklenen sürümler

| Sürüm | Durum |
|---|---|
| 1.1.0 Beta | Desteklenen beta |
| Eski beta sürümler | Yükseltme önerilir |
