# Omurga Demo İçerikleri

Bu klasör Omurga CMS ile gelen varsayılan tema demolarını içerir.

- `haber-v1`: Haber sitesi demo verileri
- `kurumsal-v1`: Kurumsal site demo verileri
- `topluluk-v1`: Topluluk/dernek demo verileri

Demo içerikler tema dosyalarından ayrı tutulur; kurulum veya tema paneli üzerinden isteğe bağlı içe aktarılmalıdır.
