# Omurga Çekirdek Blok Kütüphanesi

Bu klasörde yalnızca Omurga çekirdeğiyle gelmesi gereken temel bloklar bulunur. Haber, belediye, servis veya özel sektör blokları mümkün olduğunca paket olarak geliştirilmelidir.

## Çekirdekte kalacak ana blok grupları

- Site: Logo, Menü, Arama, Breadcrumb, Sosyal Medya
- İçerik: Metin, HTML, Görsel, Galeri, Buton
- Düzen: Satır, Kolon/Kapsayıcı, Boşluk, Ayraç
- Kullanıcı: Giriş/Kayıt, Profil, Kullanıcı Menüsü
- İçerik listeleri: Son İçerikler, Popüler İçerikler, Öne Çıkan İçerikler, Yorumlar

## Paket olarak kalması önerilenler

Manşet, sürmanşet, son dakika, nöbetçi eczane, hava durumu, namaz vakitleri, döviz, ihale, başvuru formları ve gelişmiş haber vitrini gibi alanlar çekirdeği şişirmemek için paket olarak tasarlanmalıdır.
