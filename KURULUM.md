# Omurga v1.1.0 Kurulum

1. Zip dosyasını açın.
2. `omurga-v1.1` klasörünün içindeki dosyaları hosting `public_html` içine yükleyin.
3. Boş MySQL veritabanı oluşturun.
4. Site adresini açın.
5. Kurulum ekranından veritabanı, site türü ve admin bilgilerini girin.
6. Kurulumdan sonra panel adresi: `/admin`

## Güncelleme olarak yükleme

Mevcut Omurga kurulumunun üzerine dosyaları yazdırmadan önce yedek alın.

Bu sürüm özellikle tema dosyaları, CSS ve sürüm bilgilerini günceller.
