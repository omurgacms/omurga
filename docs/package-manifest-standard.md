# Omurga Paket Manifest Standardı

Resmi paket manifest dosyası: `package.json`.

Zorunlu alanlar:

- `name`
- `slug`
- `version`
- `author`

Önerilen alanlar:

- `description`
- `min_php`
- `min_omurga`
- `main`
- `requires`
- `permissions`
- `settings`
- `admin_pages`
- `blocks`

`manifest.json` kullanılmaz. Paket sistemi tek standart olarak `package.json` okur.
