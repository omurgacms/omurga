# Omurga v3.6.6 — Sol Menü Logo Kartı Notları

Bu sürümde admin panel sol menüsünün altındaki Omurga logo kartının menü içeriklerinin üstüne binmesi düzeltildi.

## Yapılanlar

- Sol sidebar artık flex düzeninde çalışır.
- Menü alanı kendi içinde kaydırılır.
- Alt logo kartı `absolute` konumdan çıkarıldı.
- Logo kartı menünün üstünü kapatmaz.
- Daraltılmış menüde alt logo kartı gizlenir.
- Mobil menüde logo kartı normal akışta kalır.

Bu güncelleme çekirdek işleyişini değiştirmez; sadece admin panel görünüm düzeltmesidir.
