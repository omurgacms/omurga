# Omurga CMS v1.0 Beta Yayın Notları

Omurga CMS 1.0 Beta, projenin ilk genel test sürümüdür.

## Bu sürümde gelenler

- İki sade varsayılan tema:
  - Omurga Kolay: OMG tabanlı tema.
  - Omurga Sabit: PHP tabanlı tema.
- Tema güvenlik sistemi.
- OMG ve PHP tema motoru ayrımı.
- Blok, paket ve hook altyapısı.
- Form oluşturucu ve form kayıt ekranı.
- Ortak medya seçici altyapısı.
- Kurulum sonrası örnek yazı ve örnek yorum.
- MIT lisansı.
