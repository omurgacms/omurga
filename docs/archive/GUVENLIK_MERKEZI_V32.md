# Omurga v3.2 - Güvenlik Merkezi

Bu sürüm çekirdeğe site türüne özel modül eklemez. Tema düzenleyici, PHP dosya düzenleme, eklenti yükleme/silme, bakım modu, IP engelleme ve giriş güvenliği ayarlarını merkezi olarak yönetir.

## Eklenen dosyalar
- admin/security.php

## Güvenlik ayarları
- Tema Düzenleyici aç/kapat
- PHP dosyası düzenleme aç/kapat
- Eklenti ZIP yükleme aç/kapat
- Eklenti silme izni altyapısı
- Başarısız giriş limiti
- Kilit süresi
- IP engelleme listesi
- Bakım modu ve bakım mesajı

## Korunan karar
Manşet, son dakika, haber botu, duyuru, etkinlik, personel, hizmet gibi site türüne özel modüller çekirdeğe eklenmemiştir. Bu işler eklenti, tema veya blok olarak kalacaktır.
