# Omurga v3.6.6 — Profil + Dil Uyum Notları

Bu sürümde profil sistemi ile Türkçe/İngilizce dil sistemi daha net bağlandı.

## Yapılanlar

- İçerik menüsündeki ekleme ve liste bağlantıları artık aktif profile göre otomatik adlandırılır.
- Haber profili: Haberler / Haber Ekle / Haber Kategorileri.
- Kurumsal profil: Yazılar / Yazı Ekle.
- Topluluk profili: Duyurular / Duyuru Ekle / Duyuru Kategorileri.
- Boş profil: Yazılar / Yazı Ekle.
- İngilizce panelde News / Add News, Posts / Add Post, Announcements / Add Announcement karşılıkları kullanılır.
- Kategori ve etiket menüleri profile göre çevrilebilir hale getirildi.
- Post türü etiketleri dil dosyalarına bağlandı.

Çekirdeğe manşet, son dakika, haber botu veya özel post tipi eklenmedi.
