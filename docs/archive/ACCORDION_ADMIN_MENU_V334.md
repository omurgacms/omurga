# Omurga v3.3.4 - Accordion Admin Menü

Bu güncelleme admin panel sol menüsünü accordion/açılır-kapanır grup mantığına alır.

## Eklenenler

- İçerik, Tasarım, Medya, Formlar, Eklentiler, Kullanıcılar, Sistem ve Ayarlar grupları.
- Eksik görünen Statik Sayfalar bağlantısı.
- Aktif sayfanın olduğu grup otomatik açık gelir.
- Kullanıcının aç/kapat tercihi localStorage üzerinde korunur.
- Manşet / Son Dakika / Headline çekirdeğe geri eklenmedi.

## Dosyalar

- `admin/_layout.php`
- `assets/css/omurga.css`
- `bootstrap.php`
