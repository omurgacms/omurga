# Omurga Varsayılan Blok Kütüphanesi

Omurga çekirdeği, tema geliştiricilerin her temada tekrar tekrar yazmak zorunda kalmayacağı temel blokları hazır getirir. Bu kütüphane haber, kurumsal ve topluluk sitelerinde ortak kullanılan yapı taşlarını içerir.

## Çekirdek bloklar

### Site blokları

- Logo
- Menü
- Arama
- Breadcrumb
- Sosyal Medya

### İçerik blokları

- Metin
- HTML
- Görsel
- Galeri
- Buton

### Düzen blokları

- Satır
- Kolon / Kapsayıcı
- Boşluk
- Ayraç

### Kullanıcı blokları

- Giriş / Kayıt
- Profil
- Kullanıcı Menüsü

### İçerik liste blokları

- Son İçerikler
- Popüler İçerikler
- Öne Çıkan İçerikler
- Yorumlar

## Çekirdeğe alınmaması gereken bloklar

Aşağıdaki bloklar tema veya paket ihtiyacına göre ayrıca geliştirilmelidir:

- Manşet
- Sürmanşet
- TV Manşet
- Gazete Vitrini
- Son Dakika Şeridi
- Hava Durumu
- Namaz Vakitleri
- Döviz / Altın
- Nöbetçi Eczane
- Belediye ihale / duyuru / başvuru blokları

## Kural

Çekirdek bloklar her temada güvenle kullanılabilecek kadar genel olmalıdır. Belirli bir sektör veya site türüne özel bloklar paket olarak gelmelidir.
