# Omurga CMS 1.0.2 Beta Release Notes

Omurga CMS 1.0.2 Beta is a public beta release.

## What's included

- OMG template theme support
- PHP theme support
- Two default themes:
  - Omurga Kolay, OMG based
  - Omurga Sabit, PHP based
- Theme safety system
- Block system foundation
- Package system foundation
- Hook system foundation
- Media library foundation
- Form builder beta
- Secure GitHub based updater
- Automatic and manual update package flow with backups
- One sample post and one sample comment after installation
- MIT License

## Notes

This is a beta release. Test before using it on a production website and always keep a backup.
